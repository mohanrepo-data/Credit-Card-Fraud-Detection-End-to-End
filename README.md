# Credit Card Fraud Detection using Machine Learning

##  Project Overview
This repository contains a pure Machine Learning solution designed to detect fraudulent credit card transactions from highly imbalanced data. The model focuses on accurately identifying anomalies to minimize financial losses while optimizing precision to avoid false alarms.

##  Tech Stack & Libraries
- **Language:** Python
- **Libraries:** Pandas, NumPy, Scikit-Learn

## 📊 Key Insights & Model Output
- **Total Transactions Analyzed:** 57,000+
- **Actual Frauds Detected:** 98
- **Objective:** Successfully trained and evaluated a classification model to separate legitimate transactions (Class 0) from fraudulent ones (Class 1).

## 💻 How to Run
1. Clone this repository.
2. Ensure you have the required ML libraries installed (`pandas`, `numpy`, `scikit-learn`).
3. Run the python script/notebook to view the classification performance and predictions.
